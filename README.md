# PixelEdu

<div align="center">

# **The Digital Operating System for Modern Education**

### *One platform. Every classroom. Every student. Every operation.*

PixelEdu is an **AI-powered School Management System (SMS)** and **Learning Management System (LMS)** built to transform how educational institutions operate. From admissions and academics to finance, communication, analytics, and personalized learning, PixelEdu provides a unified platform designed for the future of education.

**Built for Africa. Designed for the world.**

---

</div>

## Why PixelEdu?

Most schools rely on disconnected systems, spreadsheets, paper records, and multiple third-party applications to manage daily operations. PixelEdu brings everything together into one secure, scalable, and intelligent ecosystem.

Whether you're a primary school, high school, college, or university, PixelEdu streamlines administration while improving the learning experience for students, teachers, parents, and administrators.

# Built for Real-World Connectivity

PixelEdu is designed with an **offline-first philosophy**.

Schools can continue working even with unreliable internet connectivity. Data automatically synchronizes when a connection becomes available.

# Tech Stack

| Layer          | Technology              |
| -------------- | ----------------------- |
| Frontend       | React 19 + TypeScript   |
| Framework      | TanStack Start          |
| Styling        | Tailwind CSS v4         |
| UI             | shadcn/ui               |
| Animations     | Framer Motion           |
| Backend        | Server Functions        |
| Database       | PostgreSQL              |
| Authentication | JWT + Role-Based Access |
| Deployment     | Cloud & Edge Ready      |

# Getting Started

## Requirements

* Node.js 20+
* Bun (recommended)

## Installation

```bash
git clone https://github.com/yourusername/pixeledu.git

cd pixeledu

bun install

bun dev
```

Visit:

```
http://localhost:3000
```
